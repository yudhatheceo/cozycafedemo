## 🎨 Design Highlights

- **Warm café aesthetic** — brown, cream, and sage green palette
- **Playfair Display** (headings) + **Inter** (body) typography
- **Responsive** — optimized for mobile, tablet, and desktop
- **Smooth animations** — scroll-triggered fade effects
- **Interactive menu tabs** — filter by category without page reload
- **Sticky navbar** — transparent on hero, solid on scroll

## ⚙️ Technology Stack

| Technology | Usage |
|---|---|
| **HTML5** | Semantic page structure |
| **Vanilla CSS** | Custom design system, animations, responsive layout |
| **JavaScript (ES6+)** | Navbar, scroll effects, tab filter, form handling |
| **Google Fonts** | Playfair Display + Inter |
| **JSON-LD** | Local Business structured data for SEO |

> No frameworks. No build tools. Lightweight and fast by design.

---

## 🔍 SEO Features

- Unique `<title>` and `<meta description>` per page
- Open Graph tags for social sharing (Facebook, WhatsApp)
- JSON-LD Local Business schema (`index.html`)
- Descriptive `alt` text on all images
- Lazy loading on non-critical images
- Mobile-first responsive design (Core Web Vitals friendly)
